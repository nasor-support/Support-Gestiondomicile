Idéal pour

Propriétaires, locataires, familles ou toute personne souhaitant garder une gestion claire de son logement, de ses équipements et de ses documents, sans dispersion.

Questions fréquentes

Mes données sont-elles envoyées sur un serveur ?

Non. Vos informations restent stockées localement sur votre Mac. Aucun compte n’est requis.

Comment sauvegarder ou transférer mes données ?

Utilisez les fonctions d’import / export intégrées à l’application pour sauvegarder un domicile ou le restaurer.

Puis-je tester l’application sans saisir mes vraies données ?

Oui. Le Mode Démo permet de créer automatiquement un domicile d’exemple.

Comment signaler un problème ?

Envoyez un email avec une description du problème, votre version de macOS et, si possible, une capture d’écran.

Besoin d’aide ?
